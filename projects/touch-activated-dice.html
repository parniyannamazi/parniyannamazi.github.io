Touch-Activated Electronic Dice

Project Overview

Soldered and assembled a touch-activated electronic dice on a PCB, interpreting schematic diagrams and verifying connections to build a working LED dice.

Skills Used

Soldering PCB Assembly SMT Through-Hole Schematic Reading Multimeter

How It Works

Touching the contact triggers the circuit to 'roll', and LED sequencing logic lights up the LEDs in the pattern of a random dice face. Building it meant reading the schematic, soldering 30+ surface-mount and through-hole components onto the PCB, and verifying every connection before powering it on.

Main Features

  • Soldered and assembled 30+ surface-mount and through-hole components on a PCB.
  • Interpreted schematic diagrams and verified connections before power-up.
  • LED sequencing logic that displays the rolled dice face.
  • Tested and debugged the LED logic with a multimeter to improve functionality and reliability.
