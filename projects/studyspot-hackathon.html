StudySpot

explore, share, study...smart

✨ Inspiration

Every student knows the struggle of finding a good study spot on campus. You walk from building to building only to see every table full. Our team wanted to fix that. We created StudySpot to make finding quiet and available study spaces simple, fast, and stress-free while also helping students share real-time updates with each other.

📍 What it does

StudySpot is a web app with an interactive map that highlights study spaces around campus. Users can check whether an area is busy, moderately full, or free, and instantly report what they see. These live updates help other students decide where to go without wasting time searching in person. The app is lightweight, quick to load, and easy to use on both desktop and mobile.

Built With

React Vite Flask JavaScript TypeScript HTML CSS Node.js

⚙️ Challenges we ran into

Connecting the front end and back end while keeping real-time updates smooth was harder than expected, especially with limited time. We also struggled with GitHub merge conflicts and keeping everyone's code in sync. Designing an intuitive map layout that showed enough detail without cluttering the screen took several revisions. Balancing all of this under the 24-hour deadline taught us to prioritize teamwork, communication, and quick problem solving.

🏆 Accomplishments that we're proud of

We built a working prototype from scratch in just a few days. We used React and Vite for the front end, Flask for the back end, and managed to make the system functional, clean, and responsive. We learned to collaborate effectively under pressure and used AI tools like ChatGPT and Gemini to accelerate development and debugging.

🚀 What's next for StudySpot

  • Expand and enhance the map — include more campus locations and highlight hidden study areas students often miss, with a better map design for exploring spaces across buildings.
  • Add photo uploads and visual previews — let users upload pictures of study spaces so others can see the setup, lighting, and vibe before heading there.
  • Improve accuracy and personalization — introduce verified user reports, smarter location filters, and personalized study spot recommendations based on user habits and time of day.
