StudySpot — WiE × ESSS Hackathon

Project Overview

Built a real-time web app that displays live campus study-space availability using color-coded occupancy indicators and user-submitted updates — prototyped in 12 hours as part of a 4-person team.

Skills Used

Web Development JavaScript Real-Time Data Responsive Design UI/UX Teamwork

How It Works

The app shows campus study spaces with color-coded occupancy indicators so students can see at a glance where there's room. Availability updates in real time, and users can submit their own updates to keep the data fresh. It was designed with a responsive interface so it works on phones and laptops alike.

Main Features

  • Real-time display of campus study-space availability with color-coded occupancy indicators.
  • User-submitted updates to keep availability data current.
  • Responsive interface designed for both mobile and desktop.
  • Working prototype built within 12 hours as part of a 4-person team.
