Automatic Plant Watering System

Project Overview

Developed an automated plant irrigation system that monitors soil and climate conditions and adjusts watering cycles to make plant care easier and more efficient.

Skills Used

Microcontrollers Sensors Embedded C Sensor Feedback Actuator Control System Integration

How It Works

Sensors continuously read soil moisture and surrounding climate conditions. The controller compares those readings against target thresholds and decides when the plant actually needs water, instead of watering on a fixed timer. When it does water, indicator lights show the status, and the adaptive control loop means it only uses as much water as the plant needs.

Main Features

  • Monitors soil moisture and climate conditions with live status updates.
  • Adjusts watering cycles automatically based on sensor feedback rather than a fixed schedule.
  • Indicator lights for real-time system status.
  • Reduced manual effort by 40% and water use by 15% through adaptive watering control.
