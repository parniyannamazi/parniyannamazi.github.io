Automatic Door with Facial Recognition

ENSC 100/105W — Team LA08 (Theta): Daniel, Karan, Huy, Devon, Connor, Sean, Parniyan

Skills Used

Python OpenCV Raspberry Pi Computer Vision Servo Control Sensor Integration

Problem

  • Traditional locks are hard for people with physical or cognitive challenges.
  • Managing keys is difficult when carrying heavy items.

Solution

A keyless, hands-free door system for secure and convenient access, designed for residential homes.

Functional Requirements

  • Facial recognition for access.
  • Automatic door locking/unlocking.
  • Emergency manual control.
  • Works in all lighting conditions.

Design Parameters

  • Motion sensor turns on the light.
  • Adjustable door timing.
  • Manual lock for emergencies.

Design

  • Facial recognition camera detects known faces.
  • Sends signal to lock and motor via Raspberry Pi.
  • The latch unlocks, motors open and close the door, and the latch locks again.

Software

  • Raspberry Pi 4
  • OpenCV Face Recognition

Challenges

  • Making facial recognition work in any light.
  • Difficulty in creating the door push mechanism.
  • Budget: $209.12

Results

Successfully developed an automatic door system with facial recognition.

Constraints

  • Facial recognition may not work at some angles.
  • Too many wires.
  • Potential for fraud.
  • Doesn't work during a power outage.

Future Work

  • Motion light activates only in darkness.
  • Develop a more reliable locking system.
  • Use a standard door closer.
  • Upgrade software coding.
