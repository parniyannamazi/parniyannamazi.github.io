Automatic Door with Facial Recognition

Project Overview

Collaborated in a team of 7 to design and build a Raspberry Pi–based keyless door system that uses OpenCV facial recognition to grant access.

Skills Used

Python OpenCV Raspberry Pi Computer Vision Servo Control Sensor Integration

How It Works

A camera feeds video into an OpenCV facial-recognition pipeline running on a Raspberry Pi. When it recognizes an authorized face, control logic in Python triggers a servo to unlock the door. The system also handles motion-activated lighting and a manual override, and multiple recognition models were tested to balance accuracy against speed.

Main Features

  • Raspberry Pi–based keyless entry using OpenCV facial recognition.
  • Python control logic for real-time access, motion-activated lighting, and manual override.
  • Tested multiple recognition models to optimize accuracy and speed.
  • Integrated sensors and calibrated servos through iterative debugging to improve reliability.
